• This API provides sunset and sunrise times for a given latitude and longitude. This API don't need an API key
  • I found this API at https://sunrise-sunset.org/
  • I need to provide langtitude and longitude of the location. The timezone(tzd) parameter is optional but I provided the America/New_York timezone to make the information relevant.
  • This API can be useful for someone who wants to take picture of sunrise or sunset and for someone who just really want to enjoy the beauty of sunset(me)
